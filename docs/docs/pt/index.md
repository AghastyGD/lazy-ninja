# Lazy Ninja: Gere Endpoints para Django

**Lazy Ninja** é uma biblioteca Django que automatiza a geração de endpoints com Django Ninja. Ela escaneia seus models e cria schemas Pydantic para listar, detalhar, criar e atualizar registros.

## Recursos principais

- Endpoints instantâneos a partir de models
- Schemas dinâmicos
- Hooks customizáveis
- Filtros e ordenação
- Documentação automática (Swagger/ReDoc)

## Autenticação

O módulo de auth é opcional e usa JWT. Suporta modo stateless ou stateful, login configurável e hooks.

Leia o [Guia de Autenticação](auth-guide.md).
